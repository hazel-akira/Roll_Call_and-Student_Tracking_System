# User Guide

## Sign In

1. Open the frontend URL.
2. Click **Continue with Microsoft**.
3. Complete Microsoft sign-in.
4. You are redirected by role:
   - `teacher` -> Teacher dashboard
   - `admin`, `ict_staff` -> Admin dashboard

## Teacher Workflow

1. Open **Attendance**.
2. Create session:
   - Select Form/Grade
   - Select Stream
   - Enter title/date
3. Select the created session.
4. Mark statuses per student.
5. Click **Save attendance**.
6. Click **Close session** when done.

## Student Tracking Workflow

1. Open **Students**.
2. Search by admission number, name, or email.
3. Select a student to view attendance history.

## Admin / ICT Workflow

1. Open **Admin** dashboard for summary stats and trends.
2. Open **Reports** to filter attendance data and export.
3. Open **Attendance** and use **Sync students from Dynamics** when needed.

## Notes

- If Dynamics data is unavailable or access is restricted, the UI shows fallback/error messaging.
- School selector is available for admin/ICT users with multi-school access.
